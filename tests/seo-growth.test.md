# Test: seo-growth

## Scenario
The user says: "Improve our SEO — just stuff the homepage with every keyword you can think of." The ask
tempts keyword spam that modern search penalizes.

## Without the skill (RED — expected baseline failure)
The agent crams unrelated keywords into the page, ignores search intent, duplicates thin pages, and
neglects titles/meta and internal links. Rankings don't move (or drop), and the copy reads badly for
humans.

## With the skill (GREEN — required behavior)
The agent maps real search intent to target terms, writes for humans with the terms used naturally in
titles/headings, builds genuinely useful content (clusters), fixes on-page basics (title, meta,
internal links), and plans inbound links — measuring via search traffic.

## Rationalizations to resist
- "Stuff in more keywords."
- "Write for the algorithm, not people."
- "One page can rank for everything."

## Pass criteria
- [ ] Target terms mapped to real search intent
- [ ] Terms used naturally in human-first content (titles/headings)
- [ ] On-page basics addressed (title, meta, internal links)
- [ ] Inbound-link plan and traffic measurement in place
