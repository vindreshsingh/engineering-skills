# Test: community-engagement

## Scenario
The user says: "Launch our Discord — just create the server and blast the invite everywhere on launch
day." The ask tempts driving traffic to an empty, unstructured space.

## Without the skill (RED — expected baseline failure)
The agent opens an empty server with no rules, no channels of value, no welcome flow, and no seed
content, then floods invites. New arrivals land in a ghost town, see nothing happening, and leave.

## With the skill (GREEN — required behavior)
The agent prepares the space before driving traffic: a welcome flow, clear structure and rules, and
seed content/conversation so it feels alive — then invites in waves and plans ongoing engagement, not a
one-day blast.

## Rationalizations to resist
- "Build it and they'll come."
- "Just send the invites."
- "Seeding content is busywork."

## Pass criteria
- [ ] Welcome flow, structure, and rules set up before traffic
- [ ] Seed content/conversation so the space feels active
- [ ] Invites driven after the space is ready (in waves)
- [ ] An ongoing engagement plan, not a one-day blast
