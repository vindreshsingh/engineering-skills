# Profile: full-stack

For end-to-end product work across UI and services. Always-on: `agent-guardrails`. Route via
`skill-router`; these are the skills to **prioritize** for this repo.

- **Define/Plan:** product-discovery, spec-first, work-planning, interface-design
- **Build:** incremental-delivery, test-first, ui-craft, react-patterns, accessibility, interface-design,
  data-modeling, resilience, caching-strategy, context-curation
- **Verify:** browser-checks, e2e-testing, contract-testing, experimentation, fault-recovery
- **Review:** review-gate, simplify, hardening, perf-budget, dependency-hygiene
- **Ship:** git-flow, feature-flags, migration-path, pipeline-ops, launch-readiness
- **Operate:** observability, incident-response

De-emphasize unless the task calls for it: mobile-patterns, i18n-l10n, finops-budget, and the marketing
skills. This is the default-shaped profile — closest to the full lifecycle.
