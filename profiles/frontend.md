# Profile: frontend

For UI / web-app work. Always-on: `agent-guardrails`. Route via `skill-router`; these are the skills to
**prioritize** for this repo.

- **Define/Plan:** spec-first, work-planning, ux-design
- **Build:** incremental-delivery, test-first, ui-craft, react-patterns, accessibility, design-handoff,
  micro-interactions, i18n-l10n, context-curation
- **Verify:** browser-checks, e2e-testing, experimentation, fault-recovery
- **Review:** review-gate, simplify, perf-budget, dependency-hygiene
- **Ship:** git-flow, feature-flags, pipeline-ops, launch-readiness

De-emphasize unless the task calls for it: data-modeling, caching-strategy, resilience, observability,
incident-response, finops-budget, and the marketing skills.
