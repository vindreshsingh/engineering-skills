# Social Channel Checklist

Quick reference to run alongside [[social-distribution]]. Use per platform before publishing.

## Pre-publish (all platforms)

- [ ] Hook in the first line — pain, stat, or bold claim
- [ ] One clear CTA with working UTM link
- [ ] No unverified claims; `[VERIFY]` items resolved
- [ ] Voice matches campaign brief (technical, helpful, no hype)

## X / Twitter

- [ ] Thread: one idea per tweet; last tweet = CTA + link
- [ ] Character count checked per tweet
- [ ] 0–2 hashtags max (often zero is better)
- [ ] Screenshot or GIF attached where promised

## LinkedIn

- [ ] Story arc: problem → journey → lesson → soft CTA
- [ ] Carousel: one point per slide; last slide = CTA
- [ ] No engagement bait ("comment YES if…")

## Reddit / forums

- [ ] Subreddit rules read — self-promo limits noted
- [ ] Value-first: full guide in post body where possible
- [ ] Transparent that you're the builder if relevant
- [ ] Link placement follows community rules (post vs comment)

## Launch day

- [ ] Maker story / first comment drafted (HN, PH)
- [ ] 24h blocked for replies to comments
- [ ] Reply templates ready for common questions
