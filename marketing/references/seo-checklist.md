# SEO Checklist

Quick reference to run alongside [[seo-growth]]. Use per target page and monthly review.

## On-page (per page)

- [ ] One primary keyword assigned — no cannibalization with other pages
- [ ] Title tag 50–60 chars; keyword near front
- [ ] Meta description 150–160 chars; includes CTA verb
- [ ] Single H1 matching search intent
- [ ] H2/H3 cover related sub-questions
- [ ] URL short and readable
- [ ] CTA above fold and at end
- [ ] Image alt text descriptive

## Landing page

- [ ] Value prop in H1 (what + for whom)
- [ ] Social proof (testimonials, counts — real or `[PLACEHOLDER]`)
- [ ] FAQ section targeting long-tail queries
- [ ] Open Graph + Twitter card tags
- [ ] Page loads fast (mobile-friendly, LCP rough check)

## Technical

- [ ] HTTPS everywhere
- [ ] `robots.txt` allows public pages
- [ ] XML sitemap submitted to Search Console
- [ ] No broken links on target pages
- [ ] Canonical tags on duplicates

## Content cluster

- [ ] Hub page defined with 3+ spoke topics
- [ ] Spokes link to hub; hub links to spokes and sign-up
- [ ] Spoke drafts assigned to content-marketer with keywords

## Monthly review

- [ ] Impressions, CTR, average position from Search Console
- [ ] Organic sessions and sign-ups from analytics
- [ ] Underperforming pages flagged for title/meta refresh
