# Email Nurture Checklist

Quick reference alongside [[email-nurture]].

## Sequences

- [ ] Welcome (Day 0–14) mapped with single CTA each
- [ ] Activation trigger for inactive users
- [ ] Re-engagement at 14+ days inactive
- [ ] Unsubscribe link in every email

## Copy

- [ ] Subject ≤50 chars; preview text set
- [ ] 150–300 words; one idea per email
- [ ] UTM on all links
- [ ] Plain/minimal HTML — developer-friendly tone

## Technical

- [ ] SPF, DKIM, DMARC configured
- [ ] Triggers wired to product events
- [ ] Test send to Gmail + mobile

## Metrics

- [ ] Open, click, unsubscribe tracked
- [ ] Activation cohort vs no-email baseline
