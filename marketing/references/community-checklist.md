# Community Engagement Checklist

Quick reference to run alongside [[community-engagement]]. Use before driving traffic to community.

## Setup

- [ ] Primary community home defined (in-app, Discord, forum)
- [ ] Rules pinned — respectful, on-topic, no spam
- [ ] Channels: introductions, help, showcase (or equivalents)
- [ ] Activation metric defined (e.g. first connection in 7 days)

## Before traffic arrives

- [ ] 10+ seed posts (real team intros, tips, questions — no fake personas)
- [ ] Pinned "Start here" guide — value in 5 minutes
- [ ] Mods identified; response SLA ≤4h in launch week

## Welcome flow

- [ ] Day 0 welcome (email + in-app)
- [ ] Profile completion prompt
- [ ] Suggested first action within 24h
- [ ] Day 1 nudge if inactive
- [ ] Day 7 check-in with tip or event invite

## Weekly programming

- [ ] 4-week calendar: intros, AMA, showcase, challenge
- [ ] Facilitator assigned per activity

## Outreach

- [ ] 5 external communities identified
- [ ] Value-first participation plan (no spam)
- [ ] Conversion tracking per community source

## Feedback loop

- [ ] Weekly digest template: top questions, pain points, wins
- [ ] Route to product-manager when feedback drives backlog
