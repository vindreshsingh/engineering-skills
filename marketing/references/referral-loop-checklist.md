# Referral Loop Checklist

Quick reference alongside [[referral-loop]].

## Design

- [ ] Invite moment is post-value (not sign-up screen)
- [ ] Unique referral URL per user
- [ ] Friend landing shows referrer context
- [ ] Share copy pre-written (editable)

## Incentives & abuse

- [ ] Reward requires referred user activation
- [ ] Self-referral blocked
- [ ] Monthly cap on rewards
- [ ] Incentive is access/status preferred over cash

## Launch

- [ ] In-app prompt at invite moment
- [ ] Email mention in Day 7 sequence
- [ ] Tracking: invite rate, K-factor, referral % sign-ups

## Red flags

- [ ] No reward on sign-up-only (bots)
- [ ] Link tested end-to-end
